Godot General Purpose Modules
Annabelle Lewis - 2021

This project contains a number of general purpose modules, these can be used in any project without permision, though credit is appreciated.  

# Project Structure
	These modules are created with the module-asset project structure in mind.  When using this structure your Godot project willl have two top level folders, the Modules folder, contianing general purpose modules, and the Assets folder, contining game specific objects (rooms, tilemaps, etc...)
