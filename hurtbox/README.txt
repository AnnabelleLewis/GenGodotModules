# Description
	Basic hitbox/hurtbox implimentation

# Dependencies
	None

# Impimentation
	This module contains two classes, the hitbox and the hurtbox.  When a hitbox
encounters a hurtbox the hitbox will emit the signal "take_damage" with the 
parameter "damage", which corresponds to the damage stat of the hurtbox
